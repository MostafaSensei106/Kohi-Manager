DarkTheme

DarkTheme top-level property

ThemeData DarkTheme
final

A preconfigured ThemeData for the app's dark theme.

Defines a full ColorScheme with carefully selected color values for primary, secondary, tertiary, background, surface, error, and outlines in dark mode.

This theme uses:

  • Brightness.dark
  • Transparent AppBar with no elevation
