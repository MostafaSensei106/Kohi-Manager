LightTheme top-level property

ThemeData LightTheme
final

A preconfigured ThemeData for the app's light theme.

Defines a full ColorScheme with carefully selected color values optimized for light backgrounds. This ensures proper contrast and accessibility across the app UI.

Includes:

  • Brightness.light for overall brightness
  • Transparent AppBar with no shadow for a flat and modern appearance
